macOS default keybindings
⌘=Cmd  ⇧=Shift  ⌥=Option  ⌃=Ctrl
Split
⌘D右に分割
⌘⇧D下に分割
⌘[ / ⌘]前 / 次のsplitへ
⌘⌥+矢印方向でsplit移動
⌘⇧Entersplitをズーム
Tab
⌘T新規タブ
⌘1〜9タブへ移動
⌘⇧[ / ⌘⇧]前 / 次のタブ
⌘Wsplit / タブを閉じる
Other
⌘Enterフルスクリーン
⌘⇧Pコマンドパレット
⌘K画面クリア
⌘F検索
⌘, / ⌘⇧,config編集 / reload
スマホで開いたらホーム画面に追加すると疑似アプリとして使えます。
全一覧は ghostty +list-keybinds --default で確認できます。
