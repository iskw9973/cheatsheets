ノーマルモードの基本操作
⌃=Ctrl  コマンドはノーマルモードで入力
移動
w / b次 / 前の単語へ
0 / $行頭 / 行末へ
gg / Gファイル先頭 / 末尾へ
⌃D / ⌃U半画面下 / 上へスクロール
編集
ciw単語を消して書き換え
dd / yy / p行を削除 / コピー / 貼り付け
u / ⌃Rundo / redo
.直前の変更を繰り返す
検索・置換
/foo → n / N検索して次 / 前へ
*カーソル下の単語を検索
:%s/old/new/gファイル全体を置換
バッファ（開いたファイルの裏タブ）
:ls開いてるバッファの一覧
:b foo名前で切り替え（Tab補完が効く）
:bn / :bp次 / 前のバッファへ
:bd今のバッファを閉じる
ファイル一覧（netrw）
:Exファイル一覧を開く
Enter / -開く / 上のディレクトリへ
% / d新規ファイル / ディレクトリ作成
:Rex一覧から抜けて元のファイルへ戻る
保存・終了
:wq / :q!保存して終了 / 破棄して終了
ciw と . の組み合わせが強力：1箇所書き換えて n で次へ飛び . で繰り返す。
netrw から抜けるのは :Rex（他に何も開いてなければ :q でそのまま終了）。
操作に迷ったら :Tutor で対話チュートリアルが開けます。
